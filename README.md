### 🛠 Fedora 软件转译工具箱 (Packaging Tools)

作为软件仓库维护者，我主要使用以下工具链：

| 工具名称 | 类型 | 主要用途 | 状态 |
| :--- | :--- | :--- | :--- |
| **Alien** | 转换器 | `.deb` ⇄ `.rpm` 直接转换 | 常用 |
| **FPM** | 打包机 | 多格式打包 (Effing Package Management) | 生产主力 |
| **Flatpak** | 沙盒 | 解决依赖冲突，跨发行版分发 | 核心适配 |
| **Distrobox** | 容器 | 在 Fedora 中运行 Ubuntu/Arch 环境进行编译调试 | 环境标配 |
